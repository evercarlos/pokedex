### Para la carpta publica(archivos estaticos)
- Instalar: npm i @nestjs/serve-static

# Ejecutar en desarro
1. clonar el repositorio
2. Ejecutar
```
npm install
```
3. Tener Nest CLI instalador
```
npm i -g @nest/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

### Stack usado
* MongoDB
* Nest
